Why choose this

Choosing a project like AirCanvas can be motivated by several factors. It combines various technologies like Python, OpenCV, Mediapipe, gTTS, and EasyOCR, making it a comprehensive learning experience in computer vision, natural language processing, and speech synthesis. Additionally, it offers a creative and interactive way to explore real-time text recognition and speech generation, which can be exciting and engaging for both developers and users.

Technologies Used

Python: Used as the primary programming language for development. OpenCV: Employed for computer vision tasks, such as capturing and processing video frames. Mediapipe: Utilized for hand tracking and gesture recognition, enabling writing in the air. gTTS (Google Text-to-Speech): Used to convert the recognized text into speech. EasyOCR: Applied for optical character recognition (OCR), assisting in text prediction from hand gestures.

Scientific Exploration

Computer Vision: Investigating real-time hand tracking and gesture recognition using OpenCV and Mediapipe to enable writing in the air. Natural Language Processing (NLP): Exploring text prediction and recognition of handwritten characters through EasyOCR, enhancing the user experience. Speech Synthesis: Utilizing gTTS to convert the recognized text into speech, demonstrating the integration of speech technologies with visual inputs from the camera.

Abstract

Drawing has long been an important component of the artistic process, assisting artists in planning and expressing their ideas. People used to draw on stones with charcoal or tree branches, as shown in cave paintings, to transmit information about animals. Today, we can draw on computers, smartphones, and other gadgets. The Air Canvas is an intriguing concept that allows you to create in the air without using a keyboard or mouse. It detects movement using Python and user-friendly modules such as OpenCV and MediaPipe. We intend to improve it by incorporating the EasyOCR, which will allow us to recognise words drawn in the air. The project will also include a text-to-speech feature to vocalise recognised words, increasing its versatility and accessibility.
